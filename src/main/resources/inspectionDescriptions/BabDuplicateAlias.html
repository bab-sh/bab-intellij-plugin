Reports duplicate or conflicting task aliases in babfiles.

This inspection detects aliases that conflict with existing task names, duplicate aliases within the same task, and aliases that conflict across different tasks.

Example of problems:


tasks:
  build:
    alias: b
    aliases:
      - b          # Error: Duplicate alias
  test:
    alias: build   # Error: Alias conflicts with task name
  lint:
    alias: b       # Error: Alias conflicts with alias in 'build'

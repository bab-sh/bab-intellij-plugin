Reports unresolved task references in babfile dependency lists.

This inspection checks that all task names referenced in the deps field exist as defined tasks in the same babfile. It also detects self-referential dependencies where a task depends on itself.

Example of problems:


tasks:
  build:
    deps:
      - nonexistent  # Error: Unresolved task reference
      - build        # Error: Task cannot depend on itself
    run:
      - cmd: echo "Building"
